# vault-front-homework

## Getting started

## Install

```sh
pnpm install
```

## Run

```sh
pnpm dev
```

## Run separately

### Launch vault-server

```sh
pnpm dev:vault-server
```

### Launch vault-front

```sh
pnpm dev:vault-front
```

### Launch docs (storybook)

```sh
pnpm dev:docs
```

## Build

```sh
pnpm build
```

You can also build separately:

```sh
pnpm build:vault-server
pnpm build:vault-front
pnpm build:docs
```
